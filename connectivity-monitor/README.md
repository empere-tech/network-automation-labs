# Connectivity Monitor

A Python-based ICMP connectivity monitoring tool that detects network outages using threshold-based logic and logs outage start and recovery events.

## Features
- ICMP-based reachability checks
- Threshold-based outage detection
- Timestamped outage logging
- Graceful shutdown with outage summary
- Cross-platform support (Windows, Linux, macOS)

## Requirements
- Python 3.8+

## Usage
```bash
python connectivity_monitor.py
